VR Mobile Viewer



The Mobile Viewer project is a combination of a plug in for SketchUp and a VR mobile app that allows 3D models being built in SketchUp to be sent to multiple mobile devices in real time for viewing in VR. For my part, I designed and built the mobile app in Unity, which listens for changes to a Firebase database and receives changes as JSON. It uses the JSON to update and maintain an internal data model of the SketchUp building's components and hierarchy, then reorganizes the faces based on material to created more efficient meshes for recreating the SketchUp model on mobile. A particular challenge was that the order and speed in which the data was received from Firebase could not be relied upon, so the system had to be able to handle missing information when rebuilding partially received models, and keep track of which components were still being expected.


During the Mobile Viewer project I learned about graphics considerations for mobile devices, authentication, the Unity profiler, and gain experience planning the architecture of a small app.
