Schema

Schema is a 3D massing tool in virtual reality for the HTC Vive and Oculus headsets. On this project I was responsible for porting the already started Vive version to be compatible with the Oculus. Although the Oculus and Vive have similar controllers, a particular challenge was also creating a version that was usable with the Oculus remote, a much different input device with fewer buttons and less motion control. By separating the many inputs into distinct modes and making use of the gaze pointer I was able to effectively map all the inputs in a useable way to the remote.


Additionally I assisted with the creation of various tools for creating and interacting with 3D geometry including extruding, scaling, and manipulating vertices. The scaling involved some special considerations for allowing boxes to be scaled while conforming to a minimum width and maintaining their total floor area. As part of this task I also enjoyed creating a system that calculates the new position of a point snapped to the intersections of a grid on a plane or a three dimensional grid where the grid's size and rotation in three dimensions can be customized.


Working on Schema, I was able to learn about building cross platform applications, real time networking, and dependency injection using Zenject. I gained experience working with a large, complex code base, as well as with unit testing, 3D geometry, MVVMC, command pattern, and RESTful APIs.
